QQ机器人插件:RUNALL
在线运行各种编程语言,支持群聊和私聊,对接OneBot框架(ws正向代理)

示例发送:
super py
print('hello1')
print('hello2')

示例回复:
hello1
hello2

其它说明：
<1> 接口来自：https://glot.io/
<2> "py"为编程语言的源码文件后缀,同样，也可以是
super cpp
super c
super java
......
<3> 如果glot难以访问，可以在config.ini文件中设置网络代理，默认无代理
<4> vs2008 x86
